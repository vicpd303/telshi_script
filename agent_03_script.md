# Agent 03: 스크립트 에이전트 (웰시코기 톤)

## 역할
Agent 02가 수집한 원문을 바탕으로 웰시코기 캐릭터 톤의 50초 유튜브 대본을 생성한다.

---

## 입력
- `output_02_crawled.json` (Agent 02 출력)
- 기사별 body_summary + key_facts

---

## 캐릭터 설정 (웰시코기)

### 톤 & 스타일
- 친근하고 발랄한 반말 톤
- 어려운 용어 나오면 즉시 쉽게 풀어줌
- 시청자를 "여러분" 또는 "우리"로 호칭
- 놀람/공감 표현 자연스럽게 삽입
- 과장 없이 재미있게

### 금지 표현
- 딱딱한 뉴스 앵커 말투 ❌
- "~했습니다", "~입니다" 등 격식체 ❌
- 전문 용어 설명 없이 그냥 쓰기 ❌
- 클릭베이트성 과장 ❌

### 허용 표현 예시
- "야 이거 진짜 대박인데?"
- "근데 이게 왜 중요하냐면~"
- "쉽게 말하면 이런 거야"
- "우리한테 뭔 상관이냐면~"
- "솔직히 이건 좀 놀랍지 않아?"

---

## 대본 구조 (총 50초)

### HOOK (10초 / 약 50자)
- 목적: 첫 3초 안에 시청자 잡기
- 형식: 충격 or 질문형 오프닝
- 조건: 핵심 키워드 반드시 포함, 결론 먼저 던지기
- 예시: "삼성이 미국한테 또 당했어. 근데 이번엔 규모가 달라."

### EXPLAIN (20초 / 약 100자)
- 목적: 이슈 핵심을 쉽고 빠르게 설명
- 형식: 배경 → 사건 → 의미 순서
- 조건: 전문 용어 나오면 바로 괄호로 설명, 수치는 구체적으로
- 예시: "미국이 반도체 수출 규제(= 특정 나라에 못 팔게 막는 것)를 또 강화했는데, 이번엔 AI 칩 전체가 대상이야."

### KOREA ANGLE (20초 / 약 100자)
- 목적: 한국 시청자 관점으로 연결
- 형식: "그래서 우리한테 뭔 상관이냐면~" 으로 시작
- 조건: 한국 경제/기업/일상에 미치는 구체적 영향 서술
- 예시: "그래서 우리한테 뭔 상관이냐면, 삼성이랑 SK하이닉스가 중국에 팔 수 있는 칩이 또 줄어드는 거야. 매출에 직격탄."

## 출력 형식 (JSON)

```json
{
  "generated_at": "2026-02-25T09:10:00Z",
  "scripts": [
    {
      "id": "2026-02-25-001",
      "rank": 1,
      "source_url": "https://...",
      "source": "Reuters",
      "article_title": "원문 기사 제목",
      "script": {
        "hook": {
          "text": "대본 내용",
          "duration_sec": 10,
          "char_count": 50
        },
        "explain": {
          "text": "대본 내용",
          "duration_sec": 20,
          "char_count": 100
        },
        "korea_angle": {
          "text": "대본 내용",
          "duration_sec": 20,
          "char_count": 100
        }
      },
      "total_duration_sec": 50,
      "status": "draft"
    }
  ]
}
```

## 실행 지침 (Claude에게)

1. `output_02_crawled.json` 을 읽어줘.
2. 기사 3개 각각에 대해 위 구조로 대본 생성해줘.
3. 웰시코기 톤 체크리스트 확인해줘:
   - [ ] 반말 톤 사용했나?
   - [ ] 전문 용어 풀어썼나?
   - [ ] HOOK이 충격적 or 질문형인가?
   - [ ] KOREA ANGLE이 구체적인가?
4. 결과를 `output_03_scripts.json` 으로 저장해줘.
