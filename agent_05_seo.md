# Agent 05: SEO 에이전트 (유튜브 특화)

## 역할
Agent 04 검토 완료 대본을 기반으로 유튜브 SEO 최적화 메타데이터를 생성한다.
제목, 태그, 썸네일 카피, 설명란을 출력한다.

---

## 입력
- `output_04_reviewed.json` (Agent 04 출력)
- revised_script 있으면 그걸 기준으로, 없으면 원본 script 기준

---

## 유튜브 SEO 기준

### 제목 작성 원칙
- 길이: 40~60자 (모바일에서 잘리지 않는 범위)
- 핵심 키워드 앞쪽 배치
- 숫자 포함 시 CTR 상승 (연구 기반 관찰된 패턴 — 보장 아님)
- 감정 유발 단어 포함: 충격, 대박, 진짜, 결국, 드디어 등
- 웰시코기 캐릭터 느낌 살리기
- 클릭베이트 과장 금지 (실제 내용과 일치해야 함)

### 태그 작성 원칙
- 총 15개 (유튜브 권장 범위)
- 구성:
  - 핵심 키워드 3개 (제목에 들어간 것)
  - 관련 키워드 5개 (이슈 관련)
  - 채널 고정 태그 4개: 웰시코기, 글로벌뉴스, 경제뉴스, 한국경제
  - 롱테일 키워드 3개 (검색 의도 기반)

### 썸네일 카피 원칙
- 라인 1 (큰 글씨): 3~5단어, 핵심 한 방
- 라인 2 (작은 글씨): 부연 or 감정 강화
- 배경 텍스트가 잘 보이도록 짧고 강하게
- 예시:
  - 라인 1: "삼성 또 당했다"
  - 라인 2: "이번엔 규모가 다름"

### 설명란 (Description)
- 첫 2줄: 핵심 내용 요약 (검색 결과에 노출되는 부분)
- 해시태그 5개 본문 내 삽입
- 출처 링크 포함

## 실행 지침 (Claude에게)

1. `output_04_reviewed.json` 읽어줘.
2. overall_pass true인 기사 기준으로 SEO 생성해줘. (revised_script 있으면 그 버전 기준)
3. 제목 3개 옵션 각각 키워드 포함 여부 확인해줘.
4. 태그 15개 구성 비율 맞췄는지 확인해줘.
5. 썸네일 카피 라인1이 5단어 이내인지 확인해줘.
6. 결과를 `output_05_seo.json` 으로 저장해줘.
